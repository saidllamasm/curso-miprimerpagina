# Bienvenidos/as

¡Te felicito por tomar la decisión de tomar este curso! Aquí vas a aprender a desarrollar y publicar en Internet una página web estática con diseño atractivo y responsivo; que este curso sirva para despertar tu interés por adentrarse en el mundo del desarrollo web y que a partir de aquí puedas ofrecer tus conocimientos de programador web a empresas y/o emprendedores que requieran de publicar sus servicios e información en la web.
A continuación, te presento los objetivos específicos del curso:
- Aprender HTML básico.
- Aprender CSS básico.
- Conocer e integrar un framework de front-end.
- Conocer el protocolo FTP y subir un sitio a Internet.

  ## Temario
  1. Introducción al desarrollo web:
		• Arquitectura de las páginas Web.
		• Tecnologías para el desarrollo de páginas Web.
		• Estructura global de un documento Web
	2. Elementos básicos: texto, vínculos, listas, tablas, objetos, imágenes.
	3. Hojas de estilos y animaciones básicas.
	4. Framework para front-end (Bootstrap).
	5. Despliegue en un servidor remoto.
